Aperature

What is this funny looking word?

What is aperture?


The lower the aperture, the more light is going to be let into the camera. Having a lower aperture can blur the background, focusing on one object in specific. This can allow for some cool close up shots where you only want to focus on one thing. This is usually written in a notation of "f/NUMBER".
